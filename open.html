    Mundo open source

    O que é open source?

    Nada mais é que um software que tem seu código fonte aberto a todos,
    disponibilizado com uma licença, na qual permite, modifica-ló se necessário, usar como objeto estudantil,
    ou distribui-lo gratuitamente, para qualquer pessoa, sendo assim ela pode modificar a seu gosto.


    Seguindo essa linha de raciocínio, digo que existe software open source
    que é gratuito, e sim, software open source pago !

    Seguindo essa analogia a free Software Foundation Free so fez uma citação interressante "Um software open source, não siginifica que você não precisse
    pagar por ele, ou seja, ele pode sim ser comercial, mesmo sendo open source, segue a analogia da Free Software fundations
    Assim sendo, “software livre” é uma questão de liberdade, não de preço. Para entender o conceito, pense em “liberdade de expressão”, não em “cerveja grátis”
